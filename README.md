# Official Trakkr App

This app was designed and developed to assist Elite Dangerous Commanders who are involved in `Wing Mining Mission` operations.

# Features

- Mission Tracking
- Commodity Quantities Tracking
- Commodity Configurations
- Share Evaluations
- Sexy UI
- Themes
